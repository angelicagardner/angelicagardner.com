## Privacy & Ethics

This website is designed to be as minimal and data-respectful as possible.

### 1. Cookies
This website is a static site built with Hugo. It **does not use any cookies** for tracking, advertising, or analytics. There is no cookie consent banner because it is not necessary.

### 2. Data Collection
No personal data is collected, stored, or processed by this website.
- There are no contact forms that store information in a database.
- There is no third-party tracking (e.g., Google Analytics, Facebook Pixel).

### 3. Hosting & Infrastructure
This site is hosted on Cloudflare Pages. Cloudflare may collect anonymized metadata (such as IP addresses, browser types, and timestamps) for security purposes (e.g., DDoS protection) and performance monitoring. This data is handled by Cloudflare in accordance with their privacy standards and is not used by me for any personal identification.

### 4. External Links
My portfolio contains links to third-party websites (GitHub, LeetCode, LinkedIn, etc). Once you leave my domain, please refer to the privacy policies of those respective platforms.
